# Libreria Angular
CRUD de una libreria en Angular implementada con Material y Firebase Realtime Database

## Acerca del proyecto
* Instalar dependencias utilizando ```npm install```
* Hacer build con el comando ```ng build``` y navegar a http://localhost:4200/.
